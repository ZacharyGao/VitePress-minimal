## 如何贡献

本文档使用 VitePress 搭建，文章皆为 markdown 格式位于 `docs` 文件夹下，你可以直接在 Github 上编辑该文件进行 PR 或克隆到本地修改。

如果想在本地运行本项目，请执行以下指令：

```sh
git clone https://github.com/csfive/docs
cd docs
pnpm i
pnpm run dev
```
