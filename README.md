## 如何贡献

本文档由 VitePress 搭建，文章使用 markdown 格式书写，位于 `docs` 文件夹下，你可以直接在 Github 上编辑该文件进行 PR 或克隆到本地修改。

如果想在本地运行本项目，请克隆后执行以下指令：

```
pnpm i
pnpm run dev
```
