# 学习路线

:::tip
随着欧美众多名校将质量极高的计算机课程全部开源，自学 CS 成了一件可操作性极强的事情。毫不夸张地说，只要你有毅力和兴趣，自学的成果完全不亚于你在国内任何一所大学受到的本科 CS 教育（当然，这里单指计算机专业领域，大学带给你的显然不止是专业知识）。
:::

本项目针对课程繁多不知如何选择，资料零散甚至残缺，作业难度不知深浅，不知如何开始等各种阻力，列出以下学习路线并整合相应的学习资源以供一起学习。
